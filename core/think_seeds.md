# Proactive-think seeds — what Лілі muses about on the idle timer (v0.12).
#
# One %directive per line; lines starting with "#" are comments. Format:
#   %think {topic}    — everyday grounded musing about {topic}
#   %wonder {topic}   — a curious / imaginative leap about {topic}
#
# {topic} is optional. It may be a literal ("про Віталіка") or a placeholder that
# the core expands to live state at think-time:
#   {recent} {last_thought} {thoughts} {mood} {closeness} {plan} {need} {now} {today} {user}
#
# A seed is picked at RANDOM (no immediate repeat) on every OTHER proactive think;
# the alternate thinks are free musing (no seed). Most thoughts stay SILENT (the
# dated diary); a fraction graduate to a spoken message (LUMI_THOUGHTS_SPOKEN_RATIO).
