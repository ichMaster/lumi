## Notes

  `_turn_tools` and a turn is identical to v0.17 (auto-RAG only). The `{reply, emotion, intensity}` contract
  and the `Embedder`/`VectorStore` seams are untouched.
- **No LLMClient contract-seam change:** the trusted framing rides a `trusted_text` marker the `_tool_loop`
  recognises (like the existing image-block path), so the public `reply_structured` signature is unchanged —
  no ARCHITECTURE/contract-test update was required.
- **Isolation extended over the tool:** the recall tool wraps the user-scoped `recall_moments`; the search
  runs only over the active user's vectors (pinned A↔B).
- **The "pull" complements the "push":** auto-RAG (v0.17) still surfaces relevant memory unprompted; the
  tool adds targeted (≠ message) + multi-hop search, deduped against what's already in the prompt.
