## Notes / deviations
- **Tkinter → pygame.** The spec suggested Tkinter, but `_tkinter` is absent from the uv-managed
  Python 3.14; pygame is pip-installable and works via `uv run`, so the window uses pygame. The
  spec already allows "any simple desktop viewer".
- **Launcher is `./lumi-viewer`** (not `./viewer`): a root file named `viewer` would collide with
  the `viewer/` package directory.
- The GUI window itself isn't unit-tested (needs a display); it's a thin shell over the
  fully-tested `viewer/face.py` + the tested placeholder generator.
