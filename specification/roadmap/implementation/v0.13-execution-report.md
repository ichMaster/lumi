## How to run the bridge (laptop)

1. In `.env`: `LUMI_BRIDGE=on`, `LUMI_TELEGRAM_TOKEN=…`, `LUMI_TELEGRAM_ALLOWLIST=<your id>`.
2. Run the TUI (the brain): `uv run lumi` (or the usual entry).
3. Run the two daemons (separate processes): `uv run --extra telegram python -m telegram.inbound`
   and `uv run --extra telegram python -m telegram.outbound`.
   - The TUI must be running for replies (it's the brain); the daemons buffer while it's down.
