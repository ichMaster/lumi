## What landed

**The seam (LUMI-126).** `reply()` gains an optional **text-terminal tool-loop** (`_text_tool_loop` in
`AnthropicClient`): it offers a directive's tools and terminates on a final **text** thought ending in
`ЕМОЦІЯ:`, **never `set_state`** — the thought terminal, distinct from `reply_structured`'s set_state
terminal. The `Directive` record is now table-driven (`tools` / `cap` / `surface` / `trigger` /
`instruction_from_topic` / `family` / `append_journal` / `owner_only`). `%think`/`%wonder` keep `tools=()`
and behave exactly as v0.12.

**The seeds + safety (127, 128).** New placeholders (`{ambient_news}` / `{world}` / `{last_image}` /
`{weekday}` / `{section}` + inner-life stubs), and `core/deidentify.py` — a thought-driven external query
(wiki/news/web/`%imagine`) is de-identified (proper-noun stems from the user's own facts redacted; %prompt
exempt) before it leaves.

**The gating (129).** A family directive is **absent → plain chat** unless master + its per-family flag +
owner-rights (for %prompt/%share) + the tool/sandbox are on (`_directive_enabled`). Reused by every family.

**The six families (129–134).** file (`%note` code-appends to the sandbox journal; `%review`/`%explore`;
`%journal`), wiki (`%lookup`/`%learn`), news (`%catchup`/`%brief`), image (`%gaze`; `%imagine` paid,
own sub-cap; `%share` owner-only → Telegram, bridge-off no-op), web (`%search`/`%events`), memory
(`%recall`, trusted, no de-id) + open (`%prompt`, owner-only, topic-as-instruction).

**The UX (135).** The status line surfaces the running directive (+ active tool); a gated chat-log meta line
(`LUMI_THOUGHT_SURFACE`) marks the act — both reusable by the v0.34 scheduler.

## Contracts honored

- **No contract change** — `{reply, emotion, intensity}` is untouched; the thought terminal never reaches
  `set_state` (pinned by a unit test).
- **Untrusted by default** — tool results feed back untrusted; the one exception is `%recall` (her own
  memory, trusted). External queries are **de-identified**.
- **Per-user isolation** preserved (the de-id set, the journal, every tool are user-scoped).
- **Off by default** — `LUMI_THOUGHT_TOOLS` master + a per-family flag; off → the v0.12 tool-less think is
  byte-identical.
- **No paid/network/Telegram** — Anthropic loop driven by a fake client; wiki/news mock HTTP; image gen
  stubbed; `telegram_sink` faked.
