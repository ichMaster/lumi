### LUMI-031: Biorhythm engine — deterministic sine cycles from the birth date

**Status:** completed · **Commit:** a3a62b0
**Files:** `core/biorhythm.py` (added), `tests/unit/test_biorhythm.py` (added)

- `biorhythms(birth_date, today) -> Biorhythms` — physical (23 d) / emotional (28 d) /
  intellectual (33 d) as `sin(2π·d/period)` with phase labels (high/low/rising/falling/**critical**

- `load_birth_date`/`parse_birth_date` read `Народження: DD.MM.YYYY` from `core/natal.md`
  (graceful `None`). `format_biorhythms` renders one compact Ukrainian line.

**Validation:** unit (exact values, determinism, birth-day→critical, peak→high, crossing→critical,
parse + graceful None, formatter); ruff + py_compile; all acceptance criteria pass.
