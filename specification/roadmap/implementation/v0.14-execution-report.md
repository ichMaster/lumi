## How to run it (laptop)

1. In `.env`: `LUMI_VOICE=on`, `ELEVENLABS_API_KEY=…`, `LUMI_VOICE_ID=…`.
2. `uv sync --all-extras` (installs `elevenlabs`).
3. Run the TUI (`./lumi`) — it writes Лілі's replies to `outbox.jsonl`.
4. Run the voicer: `uv run --extra voice python -m voice.voicer`.
   - It voices **only her replies** (`kind="user"` skipped), one at a time, and **skips the existing
     backlog on first run** — then speaks each new reply aloud.
