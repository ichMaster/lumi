## Phase DoD (ROADMAP §v0.2)

> After a restart Лілі recalls what was discussed in previous sessions and durable facts about you.

Met and proved by `tests/integration/test_rehydration.py` (mock model): session 1 chat → end
(writes a summary + fact) → restart → the next turn's system prompt carries the prior summary and
fact. The live paid-model behavior is exercised by the same path with `ANTHROPIC_API_KEY` set.

## Contracts pinned this phase

- `user_id` on every per-user record (`Session`, `Message`, `ShortSummary`, `LongTermFact`).
- The `ShortSummary` and `LongTermFact` record shapes (`tests/contract/test_memory_records.py`).
- The **per-user isolation invariant** at the data level (`tests/contract/test_isolation.py`),
  reinforced by isolation tests on summaries and facts.
